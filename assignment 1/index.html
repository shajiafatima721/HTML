My First Day Learning HTML – A Beginner’s Journey into Web Development

Today marks the start of an exciting journey—I wrote my first lines of HTML (HyperText Markup Language)! If you’ve ever wondered how websites are created, HTML is the very first step. It’s the backbone of every webpage you see on the internet.

What This Code Does:

(!DOCTYPE html) : Tells the browser this is an HTML5 document.

(html) : The root element of the page.

(head) : Contains meta information (like the title).

(body) : Contains the visible content.

(h1) : A heading.

(p) : A paragraph.(/p)

Key Things I Learned on Day 1

HTML is easy to start with—no complicated setup.

You can create a webpage with just a text editor and a browser.

Why HTML is Perfect for Beginners

Simple and easy to learn.

Instant results—just save and refresh the browser.

A strong foundation for CSS and JavaScript.
